```
Week/Phase 1 (Day 1 오전):
┌─────────────────────────────┐

         ▼
Week/Phase 2 (Day 1 오후~):
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ Unit 2: Menu │  │ Unit 3: Order│  │ Unit 4: Front│
```

### 병렬화 포인트
- **Unit 1 완료 후**: Unit 2, 3, 4 동시 개발 가능
- **Unit 4 (Frontend)**: API 타입 정의만 있으면 Mock 데이터로 개발 시작 가능
- **Unit 2, 3**: DB 스키마만 있으면 독립적으로 개발 가능

### 통합 포인트
- Unit 2 + Unit 4: 메뉴 조회/관리 화면 연동
- Unit 3 + Unit 4: 주문/SSE 실시간 연동
- Unit 1 + Unit 4: 인증 플로우 연동
